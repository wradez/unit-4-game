Welcome one, Welcome all! Get yo gem thang on...!

"MAKE IT SPARKLE" to begin. The rules are simple: You control the game with four crystals. Each crystal has a random value between 1-12. Clicking a crystal will add it's random value to your score. This random value is set at the beginning of each round, and will remain until a new round starts. Get your score equal to the "Achieve me" score and you win! Go over that score......SHAME!
